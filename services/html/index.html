Read:

Get all students

Endpoint: GET /api/students

On success: 200 response of an array containing all students.

On error: 500 response with a string error message.

Get a student

Endpoint: GET /api/students/[studentId: number]

On success: 200 response with the student.

On not found: 404

On error: 500 response with a string error message.

Get all students in a classroom

Endpoint: GET /api/classrooms/[classroomId: number]/students

On success: 200 response of an array containing all students assigned to the classroom.

On classroom not found: 404

On error: 500 response with a string error message.

Get all classrooms

Endpoint: GET /api/classrooms

On success: 200 response of an array containing all Classrooms.

On error: 500 response with a string error message.

Get a classroom

Endpoint: GET /api/classrooms/[classroomId: number]

On success: 200 response of the classroom.

On error: 500 response with a string error message.

Update:

Add a student

Endpoint: PUT /api/students

Form: { name=[string] }

On success: 200 response of the student's id.

On error: 500 response with a string error message.

Add a classroom

Endpoint: PUT /api/classrooms

Form: { classroomName=[string], instructorName=[string] }

On success: 200 response of the classroom's id.

On error: 500 response with a string error message.

Add a student to a classroom

Endpoint: PUT /api/students

Form: { studentId=[number], classroomId=[string]}

On success: 200 response of the mapping's id.

On error: 500 response with a string error message.
